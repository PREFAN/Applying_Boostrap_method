# Applying_Boostrap_method
This Python script demonstrates the Bootstrap resampling technique for estimating confidence intervals of a sample mean and compares it with the traditional analytical (parametric) approach. The code provides a practical and visual explanation of how bootstrapping works, making it an excellent educational tool for understanding statistical inference without relying on strong distributional assumptions.
The script begins by creating a small example dataset containing seven numerical values: [23, 45, 67, 12, 89, 34, 56]. It then performs 1000 bootstrap iterations. In each iteration, the code randomly samples from the original dataset with replacement (allowing the same value to be selected multiple times), creating a bootstrap sample of the same size as the original data. The mean of each bootstrap sample is calculated and stored. This process generates a distribution of sample means known as the bootstrap distribution.
After generating the bootstrap samples, the code calculates a 95% confidence interval using the percentile method — taking the 2.5th and 97.5th percentiles of the bootstrap means. This non-parametric approach makes no assumptions about the underlying data distribution.
For comparison, the script also computes an analytical confidence interval using the t-distribution, which assumes the data is normally distributed. It calculates the sample mean, standard error, and applies stats.t.interval() to produce the traditional parametric confidence interval.
The most insightful part of the code is the visualization. It plots a histogram of all 1000 bootstrap means, clearly showing the sampling distribution of the mean. The plot includes vertical lines marking the bootstrap confidence interval (in red dashed lines), the analytical confidence interval (in green solid lines), and the original sample mean (in black). This visual comparison helps users see how closely the bootstrap method aligns with the classical statistical approach.
Finally, the script prints both confidence intervals for direct numerical comparison.
Overall, this project effectively illustrates one of the most powerful modern statistical techniques — bootstrapping. It shows how resampling methods can be used to estimate uncertainty and construct confidence intervals even when sample sizes are small or when the underlying data distribution is unknown or non-normal. By contrasting bootstrap results with the traditional t-distribution method, the code highlights the strengths of non-parametric statistics and provides valuable intuition about sampling variability, confidence intervals, and statistical inference.
This implementation is particularly useful for students, data analysts, and aspiring data scientists who want to develop a deeper understanding of resampling techniques and their practical application in real-world data analysis.
